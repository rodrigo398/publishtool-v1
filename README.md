# PublishTool - Herramienta para posteo automatico de imagenes en FB

```
### Aclaraciones
Herramienta del tipo social media que nos permite postear imagenes automanticamente en una pagina de nuestra cuenta de facebook. Recordar armar la app dentro de la cuenta, registrarla y configurar los callback para nuestra aplicacion.

#webpack
Revisar webpack y configurarlo a gusto.

# comando para crear un cert
sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout ./certs/selfsigned.key -out ./certs/selfsigned.crt

# Lanzar web server
$ npm run dev-server
```
